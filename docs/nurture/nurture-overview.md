# Nurture Overview

Nurture is an agent that works alongside [Designer](../getting-started/what-is-designer.md) in the development workflow. While Designer focuses on creating the [Low-Level Design](../core-concepts/what-is-an-lld.md), Nurture handles activities downstream of the LLD.
