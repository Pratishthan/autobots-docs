# Login & First-time Setup

## Prerequisites

Make sure you have completed the [Prerequisites](prerequisites.md) before you login the first time.

## Logging in

1. Navigate to the Designer [login page](http://fbp-idp.ad.infosys.com:3338)
2. Log in using your **GitHub ID**

## First-time access

If this is your first time logging in, you will be prompted to **grant access** to the Designer application for your GitHub account. Approve the permissions to continue.


## After login

Once logged in, Designer will guide you to [set your context](../workspaces/setting-context.md) — selecting the repository and Jira ticket you'll be working on.
